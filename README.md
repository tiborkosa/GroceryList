# GroceryList
Android mobile app for interactive grocery list

How to use it:
    Needs google-services.json for both the main (free version) and paid version
    The package name for the paid version is com.example.grocerylist.paid under client

    If you want to use Facebook login replace the following with your own:
        facebook_application_id
        fb_login_protocol_scheme
        in the strings.xml file
